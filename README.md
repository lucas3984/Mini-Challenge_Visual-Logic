# Logica Visual - Visual Programming MVP

## Description
A functional MVP of a visual drag-and-drop programming environment, where logical blocks can be snapped together to execute actions in a gamified setting.

## Agentic Programming Context
This project is designed to practice agentic programming workflows using opencode. All development follows the guidelines in `AGENTS.md`, including code conventions, agentic task execution patterns, and project rules.

## Objectives
Develop a functional MVP of a visual programming environment (drag-and-drop style) with logical blocks that can be assembled to execute actions in a gamified environment.

## Tech Stack
- Vanilla JavaScript (ES6+ modules, no frameworks)
- Static CSS with BEM naming conventions
- Hash-based SPA routing
- State persistence via `localStorage`
- No build tools, no npm dependencies

## Prerequisites
Modern web browser (Chrome, Firefox, Edge, etc.)
